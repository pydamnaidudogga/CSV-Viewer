# CSV-Viewer

### Pages

-	Home page to upload the CSV file
-	Upload any csv file into the system
-   Display a list of all uploaded csv files on home page
-   There is a search box which searches on the front and also search each column
-	shorting options also there.When you click on the heading of a column.after that the column is short
-   validation on the front end and server side on being able to upload only csv type of files
-  Pagination of the data displayed in the table to a max of 100 records per page

## Installation

```bash
npm install
```
